## 🔄 Vybraná trasa — "Fjordy + Lofoty" (~35 dní, ~7 474 km)

Velký okruh, **bez vracení stejnou cestou** — nahoru norským pobřežím za polární kruh, zpět přes laponské Švédsko/Finsko + Pobaltí + Krakov:

> **Tachov** → Berlín → Kodaň → Oslo → Bergen (fjordy) → Geiranger/Trollstigen → Trondheim → polární kruh → 🚢 **Lofoty** → Narvik → Abisko/Kiruna → Rovaniemi → Helsinki → 🚢 Tallinn → Riga → Vilnius → Varšava → Krakov → **domů**

### 🗺️ Celý okruh na mapě

- 👀 **Náhled na GitHubu:** otevři **[trasa.geojson](trasa.geojson)** — GitHub ho vykreslí jako **interaktivní mapu** (všechny zastávky + spojnice, trajekty modře). 🟢 = základny na středy.
- 🗺️ **Do Google My Maps:** stáhni **[trasa.kml](trasa.kml)** → [My Maps](https://www.google.com/mymaps) → *Vytvořit mapu → Importovat* (My Maps umí KML/KMZ/GPX/CSV, **ne** GeoJSON).
- 🌍 **Rychlý náhled bez účtu:** přetáhni `trasa.geojson` na [geojson.io](https://geojson.io).

**🧭 Přímo v Google Maps (klikací, navigovatelné)** — 3 úseky podle dvou trajektů:

1. **[Tachov → Trondheim](https://www.google.com/maps/dir/Tachov/Berlin/Copenhagen/Goteborg/Oslo/Flam/Bergen/Geiranger/Andalsnes/Trondheim)** (DE/DK/SE + jižní Norsko a fjordy)
2. **[Trondheim → Helsinki](https://www.google.com/maps/dir/Trondheim/Mo+i+Rana/Bodo/Svolvaer/Narvik/Abisko/Kiruna/Rovaniemi/Oulu/Helsinki)** (polární kruh, Lofoty, Laponie) — *🚢 Bodø→Lofoty trajektem*
3. **[Tallinn → domů](https://www.google.com/maps/dir/Tallinn/Riga/Vilnius/Warszawa/Krakow/Tachov)** (Pobaltí + Polsko) — *🚢 Helsinki→Tallinn trajektem*

> Trasa kopíruje **reálné silnice** (vygenerováno přes OSRM), trajekty jsou modře. Ověřená silniční vzdálenost: **~7 474 km**.

**Otevřená rozhodnutí k validaci:**
- 🟦 **Nordkapp** — prodloužení z Lofot na nejsevernější bod Evropy, +~1–2 týdny (varianta C)
- 🟨 **Zkrácení** — otočka už za fjordy (varianta A, ~3 týdny), když nezbyde čas na Arktidu

## 💸 Rozpočet (orientačně, 2 lidi, ~5–6 týdnů)
**~3 200–4 800 €** všeho všudy (bez auta) — díky EV (~450–650 € elektřina) a spaní v autě zdarma (allemansrätten).

## ✨ Proč sever sedí na EV + spaní v autě
- **Nabíjení bez starostí** — Norsko je EV velmoc, rychlonabíječky všude včetně Lofot i Nordkappu.
- **Spaní zadarmo skoro všude** — allemansrätten (right to roam) v NO/SE/FI.
- **Chladné noci** = výborné spaní, Camp Mode skoro netřeba (šetří baterku).
- Pozor jen na: **dražší jídlo** (~2× vs. střední Evropa), **norské mýto/tunely** (AutoPASS) a **půlnoční slunce** (nutné zatemnění oken).
